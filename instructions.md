# Инструкция по работе с тренажёром

## Общие принципы
- Управление осуществляется **только левой кнопкой мыши**.
- Все тесты используют **пастельную цветовую палитру** для снижения зрительной нагрузки.

## Типы тестов

### 1. Простая реакция
- Нажмите и удерживайте кнопку «Старт».
- Через 0.5–2 сек появится **один стимул**.
- **Отпустите** «Старт», затем **кликните** на стимул.

### 2. Реакция выбора
- Аналогично, но появляются **2 или 3 объекта**.
- Выберите тот, что **полностью совпадает** с логикой задания.

### 3. Сложный выбор
- Перед попыткой показывается **образец** (цвет + форма).
- Среди трёх объектов только один — точная копия.

### 4. Комбинированный тест A
- Два **движущихся** объекта.
- Выберите тот, что соответствует образцу.

### 5. Комбинированный тест B
- Через 1.5–2.5 сек **роли меняются**.
- Кликайте на **новый** целевой стимул.

### 6. Слежение
- Удерживайте курсор **внутри движущегося круга**.
- Скорость движения **увеличивается каждые 2 сек**.

Удачи!
