## 1. Введение

**CognitiveMotorTests** — это цифровой сервис для оценки и развития когнитивных и моторных функций человека. Программа включает 12 специализированных тестов, оценивающих различные аспекты:

- **Внимание и концентрация**
- **Скорость реакции**
- **Пространственное мышление**
- **Когнитивная гибкость**
- **Рабочая память**
- **Многозадачность**

Программа генерирует два типа отчетов:
1. **CSV-отчет** — содержит сырые данные всех выполненных тестов
2. **PDF-отчет** — аналитический отчет с расчетными метриками и рекомендациями

## 2. Установка и запуск программы

### Требования к системе
- Операционная система: Windows 10/11, macOS 10.14+, Linux
- Python 3.8 или выше
- Минимум 4 ГБ оперативной памяти
- Разрешение экрана не менее 1024×768

### Установка
1. Скачайте архив с программой
2. Распакуйте в отдельную папку
3. Установите зависимости:
```bash
pip install -r requirements.txt
```
4. Запустите программу:
```bash
python main.py
```



## 3. Интерфейс программы

### 3.1 Главное окно
Программа имеет двухпанельный интерфейс:

**Левая панель (навигация):**
- Логотип программы
- Настройки количества попыток (1-10)
- Кнопки "Очистить результаты" и "Отчёт"
- Список всех доступных тестов сгруппированных по категориям

**Правая панель (рабочая область):**

1. Выберите тест из списка слева
2. Прочитайте инструкцию

5. По завершении нажмите "Отчёт" для генерации результатов

## 4. Тестовые модули

### 4.1 Базовые реакции

#### **4.1.1 Простая реакция**
**Цель:** Измерение времени простой сенсомоторной реакции на визуальный стимул.

**Процедура:**
1. Нажать и удерживать кнопку "СТАРТ"
2. При появлении стимула отпустить кнопку
3. Немедленно кликнуть на стимул

**Оцениваемые параметры:**
- Время простой реакции


#### **4.1.2 Реакция выбора**
**Цель:** Оценка времени реакции при необходимости выбора между стимулами.

**Процедура:**
1. Запомнить образец (форма и цвет)
2. Выбрать стимул, полностью соответствующий образцу
3. Кликнуть на правильный стимул

**Оцениваемые навыки:**
- Распознавание образов
- Принятие решений

#### **4.1.3 Сложный выбор**
**Цель:** Оценка способности к выбору среди дистракторов.

**Процедура:**
1. Запомнить сложный образец
2. Выбрать точную копию среди 3 вариантов
3. Игнорировать дистракторы

**Оцениваемые навыки:**
- Селективное внимание
- Устойчивость к помехам

### 4.2 Пространственные навыки

#### **4.2.1 Реакция на движущийся объект**
**Цель:** Оценка точности временной и пространственной реакции.

**Процедура:**
1. Наблюдать за движением объекта
2. Кликнуть на объект в момент его нахождения в целевой зоне
3. Учитывать траекторию движения

**Оцениваемые навыки:**
- Предсказание движения
- Точность реакции
- Синхронизация

#### **4.2.2 Предвидение траектории**
**Цель:** Оценка способности к экстраполяции движения.

**Процедура:**
1. Наблюдать за движением объекта 3-5 секунд
2. Объект исчезает
3. Предсказать его позицию через 1.5 секунды
4. Кликнуть в предсказанное место

**Оцениваемые навыки:**
- Экстраполяция движения
- Пространственное мышление
- Ментальное вращение

#### **4.2.3 Слежение**
**Цель:** Оценка способности к слежению за движущимся объектом.

**Процедура:**
1. Удерживать курсор внутри движущегося круга
2. Круг меняет направление и скорость
3. Скорость увеличивается каждые 2 секунды

**Оцениваемые навыки:**
- Моторный контроль
- Адаптация к изменению скорости
- Устойчивость внимания

### 4.3 Когнитивные тесты

#### **4.3.1 Переключение внимания**
**Цель:** Оценка когнитивной гибкости и переключения между задачами.

**Процедура:**
1. Начинать с кликания на круги
2. После определенного момента правило меняется
3. После изменения кликать на квадраты
4. Каждое испытание ограничено 3 секундами

**Оцениваемые навыки:**
- Когнитивная гибкость
- Переключение внимания
- Ингибиторный контроль

#### **4.3.2 Тест Струпа**
**Цель:** Измерение интерференции при обработке информации.

**Процедура:**
1. Читать слово (что написано) ИЛИ называть цвет (каким цветом написано)
2. Выбирать правильный цвет из предложенных
3. Игнорировать семантическое содержание

**Оцениваемые навыки:**
- Ингибиторный контроль
- Селективное внимание
- Скорость обработки информации

#### **4.3.3 Таблицы Горбова-Шульте**
**Цель:** Оценка переключения внимания и рабочей памяти.

**Два варианта:**
1. **С подсказкой:** следующее число подсвечивается
2. **Без подсказки:** необходимо запоминать последовательность

**Процедура:**
1. Найти числа в последовательности: красная 1 → черная 24 → красная 2 → ...
2. Кликать на числа в правильном порядке

**Оцениваемые навыки:**
- Рабочая память
- Переключение внимания
- Скорость поиска

### 4.3.3 Тест на оперативную память
 Тест основан на модели рабочей памяти Баддели, оценивая визуально-пространственный блок и центральный исполнитель.

### 4.3.4 Тест на различение размеров
Оценивает визуальную дискриминацию - способность различать небольшие различия в размерах.


### 4.3.5 Тест на различение цветов
Проверяет цветовое зрение и способность различать близкие оттенки.


### 4.4 Комбинированные тесты

#### **4.4.1 Комбинированный тест A**
**Цель:** Оценка распределения внимания между движущимися объектами.

**Процедура:**
1. Следить за двумя движущимися объектами
2. Один соответствует образцу
3. Выбрать правильный объект в нужный момент

**Оцениваемые навыки:**
- Распределение внимания
- Многозадачность
- Трекинг объектов

#### **4.4.2 Комбинированный тест B**
**Цель:** Оценка способности к быстрому переключению в динамических условиях.

**Процедура:**
1. Начинается как простой тест
2. Через 1.5-2.5 секунды правила меняются
3. Быстро переключиться на новые стимулы

**Оцениваемые навыки:**
- Адаптация к изменениям
- Динамическое переключение
- Устойчивость к неожиданностям

## 5. CSV-отчет: структура и метрики

### 5.1 Общая структура CSV-файла
CSV-файл содержит сырые данные всех выполненных тестов в табличном формате. Каждая строка соответствует одной попытке теста.

### 5.2 Общие метрики для всех тестов

| Поле | Тип | Описание | Единицы измерения |
|------|-----|----------|-------------------|
| `test_name` | string | Название теста | - |
| `latency` | float | **Время принятия решения** | секунды |
| `motor_time` | float | Моторное время | секунды |
| `total_rt` | float | Общее время реакции | секунды |
| `correct` | boolean | Правильность ответа | true/false |
| `correct_count` | integer | Количество правильных ответов | - |
| `total_trials` | integer | Общее количество испытаний | - |
| `accuracy` | float | Точность выполнения | % |
| `anticipation` | boolean | Антиципация (преждевременная реакция) | true/false |
| `delay` | boolean | Задержка реакции | true/false |
| `click_x` | float | Координата X клика | пиксели |
| `click_y` | float | Координата Y клика | пиксели |

### 5.3 Специфические метрики по тестам

#### **5.3.1 Реакция на движущийся объект**
| Поле | Описание | Интерпретация |
|------|----------|---------------|
| `distance_from_center` | Расстояние от клика до центра объекта | Чем меньше, тем точнее |
| `timing_delay` | Временная ошибка | Отрицательное - опережение, положительное - запаздывание |
| `trajectory_type` | Тип траектории | horizontal_left, vertical_down, diagonal_up и др. |
| `speed` | Скорость объекта | пиксели/секунду |

#### **5.3.2 Предвидение траектории**
| Поле | Описание | Интерпретация |
|------|----------|---------------|
| `prediction_accuracy` | Точность предсказания | 0-100%, чем выше, тем лучше |
| `prediction_time` | Время для предсказания | Фиксировано 1.5 секунды |
| `observation_time` | Время наблюдения | 3-5 секунд случайно |
| `prediction_error_px` | Ошибка предсказания | Расстояние в пикселях |
| `model_error_px` | Ошибка модели | Расстояние для идеальной модели |

#### **5.3.3 Переключение внимания**
| Поле | Описание | Интерпретация |
|------|----------|---------------|
| `avg_pre_switch` | Среднее время до переключения | секунды |
| `avg_post_switch` | Среднее время после переключения | секунды |
| `switch_cost` | Стоимость переключения | `avg_post_switch - avg_pre_switch` |
| `rule_changed` | Правило изменилось | true/false |

#### **5.3.4 Тест Струпа**
| Поле | Описание | Интерпретация |
|------|----------|---------------|
| `stroop_effect` | Эффект Струпа | Разница во времени между конгруэнтными и инконгруэнтными условиями |
| `avg_congruent` | Среднее время для конгруэнтных | секунды |
| `avg_incongruent` | Среднее время для инконгруэнтных | секунды |

#### **5.3.5 Таблицы Горбова-Шульте**
| Поле | Описание | Интерпретация |
|------|----------|---------------|
| `hit_rate_50_percent` | Скорость выполнения | Время на 50% чисел |
| `max_distance_px` | Максимальное расстояние между кликами | пиксели |

### 5.4 Интерпретация ключевых метрик

#### **Время принятия решения (latency)**
- **< 0.2 с:** Очень быстрое принятие решений
- **0.2-0.5 с:** Быстрое принятие решений
- **0.5-0.7 с:** Средняя скорость
- **> 0.7 с:** Медленное принятие решений

#### **Моторное время (motor_time)**
- **< 0.2 с:** Отличная моторная скорость
- **0.2-0.4 с:** Хорошая моторная скорость
- **0.4-0.6 с:** Средняя моторная скорость
- **> 0.6 с:** Медленная моторная реакция

#### **Точность (accuracy)**
- **90-100%:** Отличная точность
- **70-89%:** Хорошая точность
- **50-69%:** Удовлетворительная точность
- **< 50%:** Требуется улучшение

## 6. Аналитический отчет PDF

### 6.1 Структура отчета

**Страница 1: Оценка когнитивно-моторной подготовленности**
- Общий балл (0-100)
- Уровень подготовки (от Новичка до Профессионального)
- Таблица оценки по 4 категориям навыков

**Страница 2: Ключевые показатели**
- Детальные метрики по каждому навыку
- Интерпретация результатов
- Персональные рекомендации

**Страницы 3-N: Результаты по тестам**
- Детальная статистика каждого выполненного теста
- Графики и таблицы (для некоторых тестов)

**Последняя страница: Заключение**
- Сводная таблица всех тестов
- Рекомендации по тренировкам
- Информация о файлах отчетов

### 6.2 Расчетные метрики в PDF-отчете

#### **6.2.1 Общий балл подготовленности**
**Расчет:**
```
Общий балл = (Базовые реакции × 0.25) + 
             (Пространственные навыки × 0.25) + 
             (Когнитивная гибкость × 0.30) + 
             (Многозадачность × 0.20)
```

**Уровни:**
- **90-100:** Отличный
- **80-89:** Продвинутый
- **60-79:** Средний
- **40-59:** Начинающий
- **0-39:** Новичок

#### **6.2.2 Категория "Базовые реакции"**
**Включает тесты:** Простая реакция, Реакция выбора, Сложный выбор

**Метрики:**
1. **Среднее время принятия решения**
   - Расчет: среднее всех значений `latency`
   - Целевое значение: < 0.5 с

2. **Консистентность реакции**
   - Расчет: 100 - (CV × 100), где CV = стандартное отклонение / среднее
   - Целевое значение: > 80%

3. **Средняя точность**
   - Расчет: среднее значений `accuracy`
   - Целевое значение: > 90%

#### **6.2.3 Категория "Пространственные навыки"**
**Включает тесты:** Реакция на движущийся объект, Предвидение траектории, Слежение

**Метрики:**
1. **Точность предсказания траектории**
   - Расчет: среднее значений `prediction_accuracy`
   - Целевое значение: > 70%

2. **Временная точность**
   - Расчет: 100 - (среднее_абсолютное(`timing_delay`) × 100)
   - Целевое значение: > 80%

3. **Пространственная точность**
   - Расчет: 100 - (среднее(`distance_from_center`) / 10)
   - Целевое значение: > 70%

#### **6.2.4 Категория "Когнитивная гибкость"**
**Включает тесты:** Переключение внимания, Тест Струпа, Таблицы Горбова-Шульте

**Метрики:**
1. **Стоимость переключения**
   - Расчет: `avg_post_switch - avg_pre_switch`
   - Целевое значение: < 0.1 с

2. **Эффект Струпа**
   - Расчет: `avg_incongruent - avg_congruent`
   - Целевое значение: < 0.2 с

3. **Точность внимания**
   - Расчет: среднее значений `accuracy` для когнитивных тестов
   - Целевое значение: > 80%

#### **6.2.5 Категория "Многозадачность"**
**Включает тесты:** Комбинированный A, Комбинированный B

**Метрики:**
1. **Эффективность многозадачности**
   - Расчет: средняя точность выполнения
   - Целевое значение: > 80%

2. **Динамическая адаптация**
   - Расчет: на основе времени реакции в динамических условиях
   - Целевое значение: оценка > 70

### 6.3 Цветовая кодировка в отчете

- **Зеленый:** Отличные результаты (> 80%)
- **Синий:** Хорошие результаты (60-80%)
- **Оранжевый:** Удовлетворительные результаты (40-60%)
- **Красный:** Требуются улучшения (< 40%)

## 7. Расчетные метрики и интерпретация

### 7.1 Формулы расчета

#### **7.1.1 Консистентность реакции**
```
CV = σ / μ
Консистентность = max(0, 100 - (CV × 100))
```
где:
- σ = стандартное отклонение времени реакции
- μ = среднее время реакции

**Интерпретация:**
- **> 90%:** Высокая стабильность
- **80-90%:** Хорошая стабильность
- **60-80%:** Умеренная стабильность
- **< 60%:** Низкая стабильность

#### **7.1.2 Стоимость переключения внимания**
```
Switch Cost = RT_post - RT_pre
```
где:
- RT_post = среднее время реакции после переключения правила
- RT_pre = среднее время реакции до переключения

**Интерпретация:**
- **< 0.1 с:** Отличная когнитивная гибкость
- **0.1-0.2 с:** Хорошая гибкость
- **0.2-0.3 с:** Умеренная гибкость
- **> 0.3 с:** Требуется тренировка гибкости

#### **7.1.3 Эффект Струпа**
```
Stroop Effect = RT_incongruent - RT_congruent
```
где:
- RT_incongruent = время для инконгруэнтных стимулов (слово не соответствует цвету)
- RT_congruent = время для конгруэнтных стимулов (слово соответствует цвету)

**Интерпретация:**
- **< 0.1 с:** Сильный ингибиторный контроль
- **0.1-0.2 с:** Хороший контроль
- **0.2-0.3 с:** Умеренный контроль
- **> 0.3 с:** Слабый контроль интерференции

#### **7.1.4 Точность предсказания траектории**
```
Prediction Accuracy = max(0, 100 - (distance / max_distance × 100))
```
где:
- distance = расстояние между предсказанной и реальной позицией
- max_distance = максимально возможное расстояние на экране

**Интерпретация:**
- **> 80%:** Отличное предвидение
- **60-80%:** Хорошее предвидение
- **40-60%:** Удовлетворительное предвидение
- **< 40%:** Требуется тренировка пространственного мышления

### 7.2 Нормы для различных групп

#### **7.2.1 Возрастные нормы (время реакции в секундах)**

| Возраст | Простая реакция | Реакция выбора | Сложный выбор |
|---------|----------------|----------------|---------------|
| 18-25   | 0.25-0.35      | 0.45-0.60      | 0.60-0.80     |
| 26-35   | 0.28-0.38      | 0.48-0.65      | 0.65-0.85     |
| 36-45   | 0.30-0.42      | 0.52-0.70      | 0.70-0.95     |
| 46-55   | 0.35-0.48      | 0.60-0.80      | 0.80-1.10     |
| 56+     | 0.40-0.55      | 0.70-0.95      | 0.95-1.30     |

#### **7.2.2 Профессиональные нормы**

| Профессия | Ключевые навыки | Целевые показатели |
|-----------|----------------|-------------------|
| Спортсмены | Простая реакция, Предвидение | < 0.2 с, > 80% точности |
| Пилоты    | Переключение, Многозадачность | Switch cost < 0.15 с |
| Хирурги   | Моторная точность | Distance < 30 px |
| Программисты | Когнитивная гибкость | Stroop effect < 0.2 с |

### 7.3 Паттерны и их интерпретация

#### **7.3.1 Быстро, но неточно**
**Показатели:** Низкое время реакции, низкая точность
**Интерпретация:** Импульсивный стиль реакции
**Рекомендация:** Тренировать тормозной контроль, увеличивать время на принятие решения

#### **7.3.2 Медленно, но точно**
**Показатели:** Высокое время реакции, высокая точность
**Интерпретация:** Осторожный стиль реакции
**Рекомендация:** Тренировать скорость при сохранении точности

#### **7.3.3 Нестабильная производительность**
**Показатели:** Высокая вариативность времени реакции
**Интерпретация:** Проблемы с концентрацией
**Рекомендация:** Тренировать устойчивость внимания

#### **7.3.4 Хорошая база, слабая гибкость**
**Показатели:** Хорошие базовые реакции, высокая стоимость переключения
**Интерпретация:** Ригидность мышления
**Рекомендация:** Тренировать когнитивную гибкость

## 8. Программа тренировок

### 8.1 Для начинающих (уровень < 40)

**Цель:** Развитие базовых навыков
**Частота:** 4-5 раз в неделю
**Длительность:** 20-30 минут

**Программа:**
1. **День 1:** Простая реакция (15 попыток) + Реакция выбора (10 попыток)
2. **День 2:** Слежение (3 попытки) + Предвидение траектории (5 попыток)
3. **День 3:** Переключение внимания (2 попытки) + Таблицы Горбова (с подсказкой)
4. **День 4:** Повторение лучших тестов
5. **День 5:** Комбинированный тест A (3 попытки)

### 8.2 Для среднего уровня (40-70)

**Цель:** Сбалансированное развитие
**Частота:** 3-4 раза в неделю
**Длительность:** 25-35 минут

**Программа:**
1. **День 1:** Реакция выбора + Сложный выбор (по 10 попыток)
2. **День 2:** Реакция на движущийся объект + Предвидение (по 5 попыток)
3. **День 3:** Тест Струпа + Переключение внимания (по 3 попытки)
4. **День 4:** Комбинированные тесты A и B (по 3 попытки)

### 8.3 Для продвинутых (> 70)

**Цель:** Оптимизация и специализация
**Частота:** 2-3 раза в неделю
**Длительность:** 30-45 минут

**Программа:**
1. **День 1:** Специализация (выбор 3 тестов по слабым навыкам)
2. **День 2:** Комбинированные и сложные тесты
3. **День 3:** Полный комплекс (все тесты по 2 попытки)

### 8.4 Принципы эффективной тренировки

1. **Регулярность:** Лучше часто и понемногу, чем редко и много
2. **Варьирование:** Чередовать типы тестов для комплексного развития
3. **Постепенное усложнение:** Увеличивать сложность по мере улучшения
4. **Отслеживание прогресса:** Сохранять отчеты для анализа динамики
5. **Адекватный отдых:** Делать перерывы между попытками

### 8.5 Мониторинг прогресса

**Ключевые показатели для отслеживания:**
1. **Еженедельно:** Общий балл подготовленности
2. **Ежемесячно:** Прогресс по слабым категориям
3. **Ежеквартально:** Сравнение с возрастными нормами

**Рекомендуемая динамика:**
- **1 месяц:** Улучшение на 10-15 баллов
- **3 месяца:** Улучшение на 25-35 баллов
- **6 месяцев:** Стабилизация на профессиональном уровне

## 9. Приложения

### Глоссарий терминов

**Антиципация (anticipation):** Преждевременная реакция до появления стимула.

**Время принятия решения (decision time):** Время от появления стимула до начала моторной реакции.

**Дистрактор (distractor):** Отвлекающий стимул, не являющийся целевым.

**Ингибиторный контроль (inhibitory control):** Способность подавлять автоматические реакции.

**Конгруэнтность (congruency):** Соответствие между различными характеристиками стимула.

**Моторное время (motor time):** Время от начала движения до его завершения.

**Стоимость переключения (switch cost):** Увеличение времени реакции при смене задачи.

**Эффект Струпа (Stroop effect):** Задержка реакции при конфликте между семантическим и перцептивным содержанием.

Когнитивно-моторный тренажёр представляет собой комплексный инструмент для оценки и развития ключевых когнитивных и моторных функций. Регулярное использование программы позволяет:

1. **Оценить** текущий уровень когнитивно-моторной подготовленности
2. **Выявить** сильные и слабые стороны
3. **Разработать** персональную программу тренировок
4. **Отслеживать** прогресс в динамике
5. **Оптимизировать** производительность для профессиональных задач
